# Changelog

All notable changes to andW Debug Viewer will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-10-08

### Fixed
- **Critical**: readme.txt version unification (0.01 → 0.1.0)
- **Major**: Distribution ZIP configuration - removed assets/ from export-ignore to include necessary CSS/JS files
- **Minor**: load_plugin_textdomain() comment clarity improvements
- **Minor**: Enhanced wp_kses_post() usage review (determined existing esc_html() usage is sufficient)

### Changed
- **Documentation**: Comprehensive English translation of readme.txt for WordPress.org compliance
- **Documentation**: Enhanced security feature descriptions in readme.txt
- **Documentation**: Added bilingual support (English primary, Japanese supplementary)
- **Build**: Optimized .gitattributes for proper WordPress.org distribution ZIP creation

### Added
- **Documentation**: Complete README.md with features, installation, usage, and security details
- **Documentation**: Comprehensive development guidelines and WordPress.org submission status
- **Documentation**: GitHub badges for WordPress, PHP, and license compatibility

## [0.0.1] - 2024-10-02

### Added
- Initial release
- Debug log viewer with line/minute-based filtering
- REST API integration for asynchronous log operations
- Production environment safety with temporary override functionality
- Multisite support with network admin priority
- Comprehensive security implementation (nonce, capability checks, input/output sanitization)
- WordPress 6.0+ and PHP 8.1+ compatibility
- Complete internationalization support
